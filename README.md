# Tribal-Survival-

Hvordan man spiller spillet

1. Velg en av 3 tribes som er i menyen. Akuratt nå er det bare "Native American" som fungerer.
2. Velg ditt kjønn, Også trykker du på play.
3. Overlev, drikk, spis, og bygg ting

4. Spillet handler om å vokse opp som barn i en av tre historiske stammer som man kan velge: Native Americans, Africans eller Vikings. Målet er å overleve i naturen som våres ancestors, lære ferdigheter som jakt og håndverk, og delta i konflikter om territorium. Spillet fokuserer på kultur, samarbeid og utvikling, og gir spillerne mulighet til å utforske ulike aktiviteter og vokse opp i en dynamisk verden. Det går ann å dra på jakt, fisking, eller å hente vann for plassen til hvor hen folk er.
